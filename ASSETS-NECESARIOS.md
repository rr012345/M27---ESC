## Cómo subirlo a GitHub
1. Crea un repositorio nuevo en GitHub (puede ser público).
2. Copia TODO el contenido de esta carpeta (los .html, style.css, script.js, README.md
   y la carpeta assets/ ya con los 75 archivos dentro) a la raíz del repositorio.
3. Sube (commit + push).
4. En el repo, ve a Settings → Pages → Source: selecciona la rama principal (main) y
   carpeta `/ (root)` → Save.
5. GitHub te da una URL tipo `https://tuusuario.github.io/nombre-repo/` en unos minutos.
